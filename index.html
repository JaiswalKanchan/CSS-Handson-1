What is css?

  1. CSS stand for cascading style sheets
  2. CSS describe How element are to be displayed on screen, paper or in other area.
  3. CSS is not necessary. CSS saves a lot of work. It can control the layout multiple web pages at once
  4. External stylesheets are stored in css files.

CSS can be added to HTML document in 3 ways:

  • Inline- By using the style attribute inside html elements1
  • Internal- By using the style element in the head section
  • External- By using link element to link to an external css file.
  • The most common way to add CSS, is to keep the style in external css files. However, In this tutorial we will use inline and internal styles, because this is easier to Demostrtate. and easier for you to try it yourself.

when comparing css class vs ID, The difference is that css class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and it is used by Javascript.

In CSS, selectors are used to target a specific element or range of element on a web page. Once an element has been targeted,a style or a set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID .both are used to target elements which a style should be applied.
